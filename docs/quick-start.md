# 빠른 시작 가이드

OrderBean을 5분 안에 시작하는 방법을 안내합니다.

## 전제 조건

- [설치 가이드](./installation.md)를 완료했어야 합니다.
- MongoDB가 실행 중이어야 합니다.

## 1단계: 서버 시작

### 백엔드 시작

```bash
cd backend
npm run dev
```

백엔드가 `http://localhost:5000`에서 실행됩니다.

### 프론트엔드 시작

새 터미널에서:

```bash
cd frontend
npm run dev
```

프론트엔드가 `http://localhost:3000`에서 실행됩니다.

## 2단계: 테스트 계정으로 로그인

브라우저에서 `http://localhost:3000`을 엽니다.

### 테스트 계정 (시드 데이터 사용 시)

**고객 계정:**
- 이메일: `customer@test.com`
- 비밀번호: `password123`

**매장 관리자 계정:**
- 이메일: `merchant@test.com`
- 비밀번호: `password123`

### 또는 새 계정 생성

1. "회원가입" 클릭
2. 정보 입력
3. 역할 선택 (고객 또는 매장 관리자)

## 3단계: 첫 번째 주문 만들기

### 고객으로 주문하기

1. **카페 찾기**
   - 상단 메뉴에서 "카페 찾기" 클릭
   - 또는 홈페이지에서 "카페 찾기" 버튼 클릭

2. **카페 선택**
   - 카페 카드를 클릭하여 메뉴 페이지로 이동

3. **메뉴 선택**
   - 원하는 메뉴의 "담기" 버튼 클릭
   - 여러 메뉴를 추가할 수 있습니다

4. **주문하기**
   - 하단의 "장바구니" 버튼 클릭
   - 수량 조절 (필요한 경우)
   - "주문하기" 버튼 클릭

5. **주문 확인**
   - 주문 상세 페이지에서 주문 정보 확인
   - 예상 픽업 시간 확인

## 4단계: 관리자로 주문 관리하기

### 관리자 대시보드 접근

1. 매장 관리자 계정으로 로그인
2. 상단 메뉴에서 "관리자" 클릭

### 주문 상태 업데이트

1. 주문 목록에서 주문 선택
2. 상태 드롭다운에서 상태 변경:
   - `pending` → `confirmed` → `preparing` → `ready` → `completed`

### 메뉴 관리

1. 관리자 대시보드에서 메뉴 목록 확인
2. 새 메뉴 추가 또는 기존 메뉴 수정

## 주요 기능 체험

### 고객 기능

- ✅ 카페 목록 조회
- ✅ 메뉴 조회 및 주문
- ✅ 주문 내역 확인
- ✅ 주문 취소 (준비 완료 전)

### 관리자 기능

- ✅ 주문 상태 관리

- ✅ 메뉴 관리

1. **두 개의 브라우저 창 사용**
